Guide

Here are Some basic terminology Guide

Terms

These are basically the academic period where the exam structure remains the same and have same set of subjects.It can be also called as an academic year. eg. Classes,Semesters, etc.

Exam Structure

This is basically the set of exams which repeats every term. eg. unit tests,internals,semesters,etc.

Subjects

A group of Subjects will create a term and each subject will be able to give any exam from exam structure once. eg. Physics ,Chemistry etc.

Marks

These are the most basic units of this app.Marks will be there for every exam per subject.
